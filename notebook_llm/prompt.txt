#------------------------------
# V1 (GEMINI)
#------------------------------

這是一個非常實用的問題。目前 Google 的 **NotebookLM** 在生成簡報（Slides）或簡報大綱時，主要是基於你提供的來源文件進行「智慧摘要」。

雖然 NotebookLM 的界面目前還沒有提供像 PowerPoint 那樣的「頁數設定選單」，但我們可以透過**對話式引導（Prompt Engineering）**和**自訂筆記**來達成你的需求。

以下是具體的自定義技巧：

---

### 1. 如何強制生成特定的頁數（例如 20 頁）？
NotebookLM 本質上會根據內容的豐富度來決定長度，若要強制增加篇幅，你可以嘗試以下策略：

* **分段生成法：** 不要一次叫它生成「整份簡報」。你可以對它下令：「請針對來源文件，規劃一份 **20 頁**的簡報大綱，並詳細列出每一頁的標題與預計內容。」
* **指令明確化：** 在對話框輸入：「請根據來源資料生成一份詳細的簡報內容。我需要總共 20 頁，請確保內容包含詳細的背景、五個核心章節（每章節 3 頁）以及結論與建議。」
* **增加細節要求：** 如果內容不夠長，你可以要求它：「請擴展第 5 到第 10 頁的技術細節，並增加更多實際案例研究，以確保整份簡報達到 20 頁的深度。」

---

### 2. 如何要求 AI 聚焦在特定領域？
這是 NotebookLM 的強項，你可以透過以下方式引導它的注意力：

* **使用「指南」（Guide）功能：** 在 NotebookLM 的筆記本指南中，選擇「自訂回答風格」或直接在對話框輸入：「在生成簡報時，請**忽略**財務數據，並將 **80% 的篇幅聚焦在『技術架構』與『資安防護』**這兩個欄位。」
* **精確提問：** 寫下指令：「請針對來源文件中的『市場趨勢』部分，編寫 5 頁深入分析的簡報內容，其餘部分僅需簡略帶過。」
* **建立「重點筆記」：** 1. 你可以先在 NotebookLM 中自己寫一張小筆記（Saved Note），列出你最看重的幾個關鍵點。
    2. 選取該筆記與你的來源文件，然後輸入：「請結合我寫的重點筆記，產出一份以此為核心的簡報。」

---

### 💡 專業小撇步
> **目前的限制：** NotebookLM 生成的通常是「簡報大綱」或「講稿內容」，它無法直接幫你排版出一份美觀的 .pptx 檔案。
> 
> **最佳工作流：**
> 1. 在 NotebookLM 中輸入：`請針對 [主題] 生成一份 20 頁的詳細簡報大綱，重點放在 [特定領域]，每頁需包含標題與 3 個重點。`
> 2. 將產出的文字複製到 **Gamma.app** 或 **Canva AI**，這些工具可以將文字大綱一鍵轉化為設計精美的 20 頁簡報。

#------------------------------
# V2 (GPT)
#------------------------------



在使用 Notebook LLM（例如 NotebookLM）產生簡報（slides）時，其實沒有「硬性控制頁數」或「固定版型」的內建按鈕，但你可以透過**提示詞（prompt）設計**來高度客製化輸出。以下是實務上最有效的方法：

---

## 一、如何「強制接近 20 頁簡報」

AI 不會完美遵守頁數，但你可以這樣引導：

### ✅ 方法：明確結構 + 頁數限制

在提示中寫清楚：

```
請幫我產生一份簡報，共 20 頁，每頁用標題 + 3~5 個重點條列。
請依照以下結構：
1. 封面
2. 背景介紹
3-5. 問題分析
6-10. 方法說明
11-15. 案例研究
16-18. 優缺點
19. 結論
20. 未來展望
```

👉 關鍵技巧：

* 明確寫「共 20 頁」
* 指定每一頁內容範圍（避免 AI 合併頁數）
* 用「第 X 頁」格式要求（效果更穩）

例如再加一句：

```
請用「第1頁、第2頁…」格式輸出，每頁內容分開。
```

---

## 二、如何讓 AI「聚焦特定領域」

這其實是提示設計的核心。

### ✅ 方法 1：限制範圍（Scope Control）

```
請只聚焦在「商業應用」與「市場分析」，不要討論技術細節。
```

### ✅ 方法 2：指定受眾

```
這份簡報是給投資人看的，請強調市場規模、商業模式與營收潛力。
```

### ✅ 方法 3：給權重（Priority）

```
請將 60% 篇幅放在市場機會，30% 在競爭分析，10% 在技術說明。
```

---

## 三、進階技巧（讓結果更穩）

### 🔹 1. 指定格式

```
每頁請包含：
- 標題
- 3~5 個 bullet points
- 簡短說明（1 句）
```

---

### 🔹 2. 分步生成（最推薦）

不要一次生成全部，效果更好：

👉 第一步：

```
先幫我規劃 20 頁簡報的大綱（只要標題）
```

👉 第二步：

```
請依照剛剛的大綱，展開每一頁內容
```

---

### 🔹 3. 用「角色設定」

```
你是一位專業顧問（如 McKinsey 顧問），請產出條理清晰的簡報。
```

---

## 四、範例完整 Prompt（可直接用）

```
請幫我產生一份 20 頁的簡報，主題是「生成式 AI 在零售業的應用」。

要求：
- 共 20 頁，每頁用「第X頁」標示
- 每頁包含：標題 + 3~5 個重點
- 聚焦在商業價值與市場機會
- 目標受眾是企業高階主管
- 避免過多技術細節

結構：
1. 封面
2. 趨勢概述
3-6. 市場機會
7-10. 應用場景
11-14. 成功案例
15-17. 挑戰與風險
18-19. 策略建議
20. 結論
```

---

## 五、重要現實提醒

* AI「無法 100% 保證剛好 20 頁」，但上述方法可大幅提高準確度
* 如果頁數不對 👉 再補一句：

  ```
  請修正為剛好 20 頁
  ```
* NotebookLM 特別依賴「你提供的資料」，資料越清楚，輸出越穩
